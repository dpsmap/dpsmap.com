  • ဌာနမွဴး
  • ျပည္သူ႔ဆက္ဆံေရးႏွင့္ ျပန္ၾကားေရးဌာန
  • ရန္ကုန္ၿမိဳ႕ေတာ္စည္ပင္သာယာေရးေကာ္မတီ
    • ရက္စြဲ ။ ။ ၂၀၁၈ ခုႏွစ္၊ မတ္လ၊ (၂၁) ရက္။

အေၾကာင္းအရာ ။ ။ ျပည္သူ႔ဆက္ဆံေရးႏွင့္ ျပန္ၾကားေရးဌာနႏွင့္ Design Printing Services Co., Ltd. တို႔ ပူးေပါင္း၍ Yangon GIS Data ျပဳစုရန္ အဆိုတင္ျပျခင္း။

အထက္အေၾကာင္းအရာပါ ကိစၥႏွင့္ပတ္သက္၍ ရန္ကုန္ၿမိဳ႕ေတာ္ စည္ပင္သာယာေရးေကာ္မတီ ျပည္သူ႔ဆက္ဆံေရးႏွင့္ ျပန္ၾကားေရးဌာနမွ Pilot Project အျဖစ္ ေဆာင္ရြက္လ်က္ရွိေသာ

    ၁။ သာေကတ Household Data Collection အတြက္ GIS Data မ်ား အတူပူးေပါင္းေကာက္၍ 360˚ Camera, Drone တို႔ျဖင့္ ပံ့ပိုးေပးပါမည္။ ပထမအဆင့္အေနျဖင့္ DPS မွ Google Earth တြင္ ျမင္ရေသာ အိမ္ေခါင္မိုးမ်ားေပၚတြင္ Household Detail မ်ား တင္ေပးၿပီး Arc GIS, Q GIS, Mapinfo Format မ်ား ျပဳစုပါမည္။ ျပဳစုခ်ိန္တြင္ ဌာန၀န္ထမ္းမ်ား Capacity Building အား On Job လက္ေတြ႔သင္ၾကားေပးသြားပါမည္။ (ပူးတြဲ-၁) ေအာက္ပါကိရိယာမ်ား အသံုးျပဳသင္ၾကားပါမည္။
      (၁) GPS
      (၂) 360 ° Camera
      (၃) GPS Video Camera
      (၄) Mobile Phone

    ၂။ ေကာ္မတီ၏ Website တြင္ ေျမပံုမ်ားအား ေအာက္ပါအတိုင္း အဆင့္ျမင့္ Software မ်ား အသံုးျပဳေရးဆြဲေပး ပါမည္။
      (၁) Arc GIS Online + Shape File
      (၂) Carto + Layers
      (၃) Elegent Themes From Divi

      အသံုးျပဳႏိုင္ပံု
      (၁) ၾကည့္သူမွ Layer မ်ား ဖြင့္/ပိတ္ ႏိုင္ပါမည္။
      ဥပမာ - YBS လမ္းေၾကာင္းႏွင့္ ကားမွတ္တိုင္သီးသန္႔
      - EO ႐ံုးမ်ားသီးသန္႔
      - အစိုးရ႐ံုးမ်ား
      - ရဲ၊ စည္ပင္၊ တရား႐ံုး၊ စုေပါင္း႐ံုး
      (၂) Vector File ကို Zoom in Zoom out ကို 3000 Scale အထိရပါမည္။
      (၃) Divi Wordpress ျဖစ္၍ ဌာနမွ Admin Control ရွိၿပီး အခ်က္အလက္ထည့္သြင္းမႈကို Log In Password ႏွင့္ ထိန္းႏိုင္ပါမည္။

      ၃။ Capacity Building အတြက္ ျပည္သူ႔ဆက္ဆံေရးႏွင့္ ျပန္ၾကားေရးဌာန၀န္ထမ္းမ်ား၊ ေျမပံု၊ GIS၊ ပံုႏွိပ္၊ Website ျပဳျပင္ေရးဆြဲနည္းပညာမ်ားအား Design Printing Services Co., Ltd. ႐ံုးတြင္ သင္တန္းဖြင့္လွစ္ သင္ၾကားေပးပါမည္။

      ၄။ ရန္ကုန္ၿမိဳ႕ေတာ္စည္ပင္သာယာေရးေကာ္မတီမွ လႊင့္တင္ထားေသာ Website တြင္ ေျမပံုပိုင္းဆိုင္ရာ အဆင့္ျမွင့္ တင္ႏိုင္ရန္ကိစၥ ေဆာင္ရြက္ေပးပါမည္။

      ၅။ Data ပိုင္ဆိုင္မႈအေနျဖင့္ ယေန႔အထိရွိေသာ ေျမပံုႏွင့္အခ်က္အလက္မ်ားမွာ Design Printing Services Co., Ltd. ၏ ပိုင္ဆိုင္မႈမ်ားျဖစ္ၿပီး ျပည္သူ႔ဆက္ဆံေရးႏွင့္ ျပန္ၾကားေရးဌာနမွ ပံ့ပိုးေပးေသာ Data မ်ားမွာ ဌာနအပိုင္ Data မ်ား ျဖစ္ေၾကာင္း အသိအမွတ္ျပဳ ေဆာင္ရြက္သြားႏိုင္ပါရန္တို႔ကို အဆိုျပဳတင္ျပအပ္ပါသည္။အဆိုပါကိစၥမ်ား လာေရာက္ရွင္းလင္းျပသႏိုင္ရန္ လူႀကီးမင္းသင့္ေတာ္သည့္အခ်ိန္တြင္ လာေရာက္ေတြ႔ဆံုလိုပါ သျဖင့္ သင့္ေတာ္မည့္အခ်ိန္ကို အေၾကာင္းျပန္ၾကားေပးေစလိုပါသည္။
